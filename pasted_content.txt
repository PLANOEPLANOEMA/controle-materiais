CONTEXTO Esse controle será usado diariamente dentro da obra para registrar e acompanhar os materiais disponíveis no local. Preciso de uma solução prática, visualmente organizada e fácil de atualizar todos os dias. OBJETIVO Criar uma nova aba dentro do site chamada “Controle de Materiais” (ou “Materiais da Obra”), integrada ao menu já existente, para que eu possa: - cadastrar materiais manualmente; - visualizar todos os materiais registrados; - atualizar essas informações diariamente; - gerar um relatório no final do dia para impressão. FUNCIONALIDADES OBRIGATÓRIAS 1. NOVA ABA NO SITE - Adicionar uma nova aba/página no menu principal do site. - Nome sugerido: “Controle de Materiais”. - Essa nova aba deve seguir o padrão visual do site já existente. 2. FORMULÁRIO DE CADASTRO DE MATERIAIS Criar um formulário funcional com os seguintes campos: - Nome do material - Quantidade - Especificações / descrição - Foto do material (upload de imagem) - Data de atualização (automática, mas pode ser visível) 3. LISTAGEM DOS MATERIAIS Após o cadastro, os materiais devem aparecer em uma listagem organizada na mesma página ou em uma seção logo abaixo. Cada item deve exibir: - Nome do material - Quantidade - Especificações - Foto - Data da última atualização 4. AÇÕES NOS REGISTROS Cada material cadastrado deve ter: - botão de editar - botão de excluir 5. RELATÓRIO FINAL DO DIA Preciso de um botão para gerar um relatório dos materiais cadastrados/atualizados no dia. Esse relatório deve: - ficar organizado para impressão; - ter layout limpo e profissional; - ser compatível com folha A4; - exibir pelo menos: - nome do material - quantidade - especificações - foto do material (se possível) - data de atualização 6. IMPRESSÃO - Incluir botão “Imprimir Relatório” - O layout de impressão deve esconder elementos desnecessários, como botões e campos do formulário - O relatório deve ficar apresentável para ser entregue ao pessoal da obra FUNCIONALIDADES EXTRAS DESEJADAS Se possível, incluir também: - campo de busca por nome do material - filtro por data - organização por ordem de cadastro ou atualização - layout responsivo - mensagens visuais de sucesso ao cadastrar, editar ou excluir - armazenamento local ou persistência dos dados, dependendo da estrutura atual do site ESTRUTURA VISUAL SUGERIDA A aba pode ser dividida em 3 blocos: 1. Formulário de cadastro 2. Lista/tabela de materiais cadastrados 3. Área de relatório com botão de gerar/imprimir REQUISITOS TÉCNICOS - Vou enviar os arquivos atuais do meu site. - Quero que você faça as alterações diretamente com base nesses arquivos. - Não quero apenas explicação, quero a implementação pronta no projeto. - Se precisar criar ou alterar HTML, CSS e JavaScript, faça isso já integrado ao site existente. - Se o site usar outro framework ou estrutura específica, adapte a solução a ele. - Preserve o restante do site funcionando normalmente. - A nova aba deve estar totalmente integrada ao projeto atual.
